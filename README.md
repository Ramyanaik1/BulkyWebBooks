# BulkyWebBooks

## Overview

BulkyWebBooks is a Book Store Management System developed using ASP.NET Core MVC, Entity Framework Core, and SQL Server. The application provides product management, category management, shopping cart functionality, order processing, and user authentication.

## Features

- User Registration and Login
- Role-Based Authorization
- Product Management
- Category Management
- Shopping Cart
- Order Management
- Company Management
- Email Notifications
- Responsive User Interface

## Technologies Used

- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- Bootstrap
- Razor Pages
- Identity Framework

## Project Structure

- BulkeyWeb.Data – Database Layer
- BulkyWeb.Models – Models and ViewModels
- BulkyWeb.Utilities – Utility Services
- BulkuWebBooks – Main MVC Application

## Installation

1. Clone the repository
2. Open the solution in Visual Studio
3. Configure SQL Server connection string
4. Run Entity Framework migrations
5. Start the application
